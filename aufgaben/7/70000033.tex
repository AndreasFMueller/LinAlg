Rechnen Sie nach, dass die Zusammensetzung der Drehung um $90^\circ$ mit der
Scherung parallel zur $x$-Achse um eine Einheit, also von
\[
Q
=
\begin{pmatrix*}[r]
0&-1\\
1& 0
\end{pmatrix*}
\qquad\text{und}\qquad
R
=
\begin{pmatrix}
1&1\\
0&1
\end{pmatrix}
,
\]
die Matrix
\[
A
=
\begin{pmatrix*}[r]
1&-1\\
1& 0
\end{pmatrix*}
\]
ergibt.


\begin{loesung}
\begin{align*}
RQ
&=
\begin{pmatrix}
1&1\\
0&1
\end{pmatrix}
\begin{pmatrix*}[r]
0&-1\\
1& 0
\end{pmatrix*}
=
\begin{pmatrix*}[r]
1&-1\\
1& 0
\end{pmatrix*}
=
A.
\qedhere
\end{align*}
\end{loesung}
