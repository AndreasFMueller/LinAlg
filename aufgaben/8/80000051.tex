Die symmetrische Matrix
\[
A
=
\begin{pmatrix*}[r]
1&2\\
2&1
\end{pmatrix*}
\]
kann mit einer Drehmatrix diagonalisiert werden.
Bestimmen Sie den Drehwinkel.

\begin{loesung}
\begin{align*}
\cot 2\alpha
&=
\frac{a_{22}-a_{11}}{2a_{12}}
=
\frac{1-1}{2\cdot 2}
=
0
&&\Rightarrow&
2\alpha
&=
\frac{\pi}2
&&\Rightarrow&
\alpha
&=
\frac{\pi}4.
\end{align*}
Die Matrix $A$ wird durch eine Drehung um $45^\circ$ diagonalisiert.
\end{loesung}
